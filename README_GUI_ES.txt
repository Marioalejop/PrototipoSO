
GUI en Español — Instrucciones
==============================
1) (Opcional) Crear venv y activarlo.
2) Ejecutar la GUI:
   python main.py --gui

Uso rápido desde los botones:
- Archivo: escribir/ver/borrar usando los campos "Archivo" y "Texto".
- Proceso: ejecutar proceso (nombre), ver lista (Procesos) y terminar por PID.
- Sistema de archivos/memoria: Listar, Formatear, Memoria (también refresca el grid).

Comandos en español aceptados (alias):
- ayuda -> help
- listar/lista -> ls
- ver/mostrar -> cat
- escribir -> write
- borrar/eliminar -> rm
- formatear -> formatear
- ejecutar/crearproceso -> run
- procesos -> ps
- terminar -> kill
- memoria -> memstat
- salir -> exit/quit

Nota: si algún comando no existe en tu shell original, verás "Comando no encontrado".
